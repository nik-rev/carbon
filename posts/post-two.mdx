## Code Blocks

Showcases basic code blocks usage

### Regular Code Block

An example of a normal code block.

```ts
const myThing = "hello world";

// a comment
function whatever() {
  return "something";
}
```

### Line numbers and filename

I can add line numbers too

```ts title="index.ts" showLineNumbers
const myThing = "hello world";

// a comment
function whatever() {
  return "something";
}
```

### Highlighted

We can highlight some words and also lines of code

```ts {2-4} /const/ /whatever/
const myThing = "hello world";

// a comment
function whatever() {
  return "something";
}
```

### Code Block wih Diffs

Added and removed lines of code

```ts
const myThing = "hello world";
const myCoolFunction = (a) => Number(a); // [!code ++]
// [!code ++]
console.log("Oh yes"); // [!code ++]

// a comment // [!code --]
function whatever() {
  // [!code --]
  return "something"; // [!code --]
} // [!code --]
```
